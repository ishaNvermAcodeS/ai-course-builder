# AI Course Builder – MVP Blueprint

## Project Description
AI Course Builder is a web application that generates a complete learning course from a user prompt.

## Target User
Students or self-learners who want a structured course quickly.

## Input
User prompt:
"I want to learn Machine Learning in 6 weeks"

## Output
- Course title
- Course description
- Weekly modules
- Lessons
- Quiz

## Core Features
1. Course syllabus generation
2. Lesson generation
3. Quiz generator
4. Course saving
5. Course export
6. AI Tutor
7. Youtube videos retrieval for courses
8. Web resources retrieval for courses

## Tech Stack
Frontend: Next.js  
Backend: FastAPI  
Database: Supabase  
AI: Groq API
